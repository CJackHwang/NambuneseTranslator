# **《正语日新标 v5.2》规范**
---

## **前言**

正语源于粤语，但其作为粤语分支，在长期独立演化过程中部分发音发生了变化，现代正语已难以同粤语共用一个标准规范。为促进**南武民主共和国**民族团结与各地区交流，建设伟大"赛博乌托邦"社会，现政宣处联合文教协会以本部都、広府地区、海実特别施政区的语言及东洋地区书写体系为参考，对语言进行系统性整理并规范推广。正语是南武集团强制性使用的标准语言。

---

## **第一章：核心原则**

### **1. 语义分工原则（名词锚点法）**

#### **汉字书写范围**
- **专有名词**：人名、地名、机构名
  - 示例：`张三`、`香港`、`中环`、`政府`、`医院`
- **普通名词**：具体事物、抽象概念
  - 示例：`飯`、`水`、`车`、`書`、`屋`、`电话`、`时间`、`方法`、`问题`、`机会`
- **基础代词**：`我`、`你`、`佢`、`我哋`、`你哋`、`佢哋`
- **数词**：`一`、`二`、`三`、`十`、`百`、`千`、`万`

#### **假名书写范围**
- **动词**：`しっ`(食)、`へおい`(去)、`らい`(来)、`たい`(睇)、`やう`(有)、`はい`(系/喺)
- **形容词**：`ほう`(好)、`だーい`(大)、`さい`(细)、`ど`(多)、`れん`(靓)
- **副词**：`ほう`(很)、`どう`(都)、`ぞん`(仲)、`しん`(先)、`ざう`(就)
- **量词**：`ご`(個)、`ぜっ`(只)、`どう`(度)、`つぃ`(次)、`やっ`(日)
- **助词/语气词**：`ぞ`(咗)、`がん`(紧)、`げ`(嘅)、`あ`(啊)、`ら`(啦)
- **连词**：`とん`(同)、`わーっぜ`(或者)、`やんわい`(因为)、`そい`(所以)

#### **指示代词特殊规则**
- **正式场合**：使用汉字
  - `呢個`、`嗰個`、`边個`、`呢度`、`嗰度`、`边度`
- **非正式场合**：可用假名（参考日语「これ/それ」）
  - `にご`、`ごご`、`びんご`、`にどう`、`ごどう`、`びんどう`

---

### **2. 音韵简化原则**

| 项目 | 规则 | 示例 |
|---|---|---|
| aa 长元音 | 使用长音符「ー」作为あ延长 | 家gaa→がー / 三saam→さーん |
| ng-声母 | 按零声母处理 | 我ngo5 → お |
| n-/l-声母 | 严格区分 | 你nei5→ねい / 李lei5→れい |
| -m/-n/-ng韵尾 | 统一写作ん | 心sam/山san/生sang → さん |
| -p/-t/-k入声 | 统一写作っ | 湿sap/失sat/塞sak → さっ |

---

### **3. 视觉连贯原则**

- **词素之间不使用空格**，保证发音连贯性和视觉流畅度

## **第三章：转写核心规则**

### **规则1：基础拆分公式**

#### **有声母音节**
```
声母 + 韵母首元音 → 第一假名
长元音ああ → あー 简化
韵母剩余部分 → 后续假名

示例：
aa（阿）＝ aa ＝あー
gaau (交) = g + aa + u = が + ーう = がーう
saam (三) = s + aa + m = さ + ーん = さーん
nei (你) = n + ei = ね + い = ねい
```

#### **零声母音节**
```
韵母完整转写

示例：
aa (阿) = あ + ー = あー
oi (哎) = お + い = おい
ng (吴) = ん
```

#### **语气词简写**
```
句末语气词可简写：
aa (啊) → あ (语气词)
aa (阿) → あー (名词，如"阿公")
```

### **规则3：关键转写规则**

#### **A. t-/c- 声母颚化**

**t- 声母：**
- 颚化：`ti → ち`、`tyu → ちゅ`
- 非颚化：`ta → た`、`te → て`、`to → と`

**c- 声母：**
- **全部不颚化**，使用つ + 小假名
- `ca → つぁ`、`ci → つぃ`、`ce → つぇ`、`co → つぉ`、`cyu → つゆ`

#### **B. d- 声母单元音规则**

- `di → ぢ`（单元音）
- `du → づ`（单元音）
- `dei → でい`（复元音拆分）
- `dou → どう`（复元音拆分）

#### **C. yu 韵母拗音规则**

**允许拗音的声母**：
- **b/p/g/k/d/t/n/l/z/s/h** → びゅ/ぴゅ/ぎゅ/きゅ/ぢゅ/ちゅ/にゅ/りゅ/じゅ/しゅ/ひゅ
- 示例：决kyut3 → きゅっ

**不允许拗音**：
- j- 视为零声母：`jyu → ゆ`
- c- 不颚化：`cyu → つゆ`

#### **D. j- 声母特殊规则**

**i/yu 韵母前视为零声母：**
- `ji → い`（医ji1→い）
- `jyu → ゆ`（鱼jyu4→ゆ）

**其他韵母前保留：**
- `jaa → やー`（也jaa5→やー）
- `jat → やっ`（日jat6→やっ）
- `je → いぇ`（也je5→いぇ）

#### **E. 鼻音/入声韵尾统一**

- **-m/-n/-ng → ん**（三saam1/山saan1/生saang1 → さーん）
- **-p/-t/-k → っ**（湿sap1/失sat1/塞sak1 → さっ）

## **第四章：书写规范**

### **1. 汉字标准**

**使用日本新字体**：気、広、学、体、国、会、経、実

**例外**：粤语特有字保持原形（冇、嘅、咗、佢）

---

### **2. 标点符号**

| 符号 | 用法 | 示例 |
|:---:|:---|:---|
| **。** | 句号 | 我へおい学校。 |
| **、** | 逗号/顿号 | 佢しっ飯、たい戏、はーん街。 |
| **？** | 问号 | 你はいびんどう？ |
| **！** | 感叹号 | ほうれん！ |
| **「」** | 引号 | 佢讲「ほうれん」。 |
| **『』** | 书名号 | 我たい『红楼梦』。 |

---

### **3. 小写假名使用原则**

#### **优先标准假名**
```
✅ di→ぢ（不写でぃ）
✅ du→づ（不写どぅ）
✅ si→し（不写すぃ）
✅ ti→ち（不写てぃ）
```

#### **必用小写假名**
```
✅ c-系列：つぁ/つぃ/つぇ/つぉ/つゆ
✅ f-系列：ふぁ/ふぃ/ふぇ/ふぉ
✅ 特殊组合：いぇ/ぐえ/くえ/ぐを/くを
```

#### **允许拗音**
```
✅ syu→しゅ / zyu→じゅ / tyu→ちゅ / dyu→ぢゅ
✅ nyu→にゅ / lyu→りゅ / hyu→ひゅ
❌ cyu→つゆ（不写ちゅ）
❌ jyu→ゆ（j-视为零声母）
```

---

### **4. 词间分写**

**基本原则**：词素间不使用空格

```
✅ 我へおい学校。
✅ 佢しっぞ飯めいあ？
❌ 我 へおい 学校。
```

## **第五章：综合示例**

### **1. 基础对话**

| 粤语 | 日新标 v5.2 | 分析 |
|:---|:---|:---|
| 你食咗飯未啊？ | 你しっぞ飯めいあ？ | 你(代词) + しっ(食) + ぞ(咗) + 飯(名词) + めい(未) + あ(啊) |
| 我今日好忙啊。 | 我今日ほうもんあ。 | 我(代词) + 今日(名词) + ほう(好=很) + もん(忙) + あ(啊) |
| 呢個系乜嘢嚟㗎？ | 呢個はいまっいぇらいがー？ | 呢個(指示词) + はい(系) + まっいぇ(乜嘢) + らい(嚟) + がー(㗎) |

（注意：原拼gaa-がー，因语气简化可为ga-が；所有长音aa结尾的语气词一并简化，可参考“啊”）

---

### **2. 复杂句**

**粤语**：听日我同你去中環睇电影、食飯、行街、买嘢。

**日新标**：听日我とん你へおい中環たい电影、しっ飯、はーん街、まーい嘢。

**分析**：
- 听日 = 时间名词（汉字）
- 我とん你 = 代词 + 连词 + 代词
- へおい = 去（动词）
- 中環 = 地名（汉字）
- たい/しっ/はーん/まーい = 动词（假名）
- 电影/飯/街/嘢 = 名词（汉字）

---

### **3. 形容词句**

**粤语**：呢间屋又大又靓，好舒服。

**日新标（正式）**：呢がーん屋やうだーいやうれん、ほうしゅふっ。

**日新标（非正式）**：にがーん屋いうだーいいうれん、ほうしゅふっ。

**分析**：
- 呢がーん/にがーん = 呢 + 间（指示词 + 量词）
- 屋 = 名词（汉字）
- やう = 又（副词）
- だーい/れん = 大/靓（形容词）
- ほう = 好（副词"很"）
- しゅふっ = 舒服（形容词）

---

### **4. 疑问句**

**粤语**：佢喺边度做乜嘢？

**日新标（正式）**：佢はい边度ぞうまっいぇ？

**日新标（非正式）**：佢はいびんどうぞうまっいぇ？

**分析**：
- 佢 = 代词（汉字）
- はい = 喺（动词）
- 边度/びんどう = 疑问词（正式/非正式）
- ぞう = 做（动词）
- まっいぇ = 乜嘢（疑问词）

---

### **5. 长句**

**粤语**：因为天気太热，所以我哋决定唔去行山，留喺屋企睇戏。

**日新标**：やんわい天気たーいいっ、そい我哋きゅっぢんんへおいはーん山、らうはい屋企たい戏。

**分析**：
- やんわい/そい = 因为/所以（连词）
- 天気/山/屋企/戏 = 名词（汉字）
- たーい = 太（副词）
- いっ = 热（形容词）
- 我哋 = 代词（汉字）
- きゅっぢん = 决定（动词）
- ん = 唔（否定）
- へおい/はーん/らう/はい/たい = 动词（假名）

## **第七章：与日语假名的差异**

| 日语规范 | 正语日新标 v5.2 | 理由 |
|:---|:---|:---|
| ぢ/じ 同音合并 | **区分使用** | ぢ=di（d-声母）/ じ=zi（z-声母） |
| づ/ず 同音合并 | **区分使用** | づ=du（d-声母）/ ず=zu（z-声母） |
| を 限用助词 | **用于w+o组合** | を=wo（蜗wo1），无助词功能 |
| ゐ/ゑ 废止 | **不使用** | 粤语无对应音 |
| 拗音限定搭配 | **扩展使用** | 允许つぁ/つぃ/つぇ/つぉ/つゆ/いぇ等 |
| 汉字用于动词 | **汉字仅用于名词** | 名词锚点法核心原则 |
| 指示代词简化 | **参考实施** | 非正式场合可用假名（にご/ごご） |
